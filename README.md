RandomContact
=============

基于android实现的随机生成联系人的工具，主要用于测试使用

#主要功能如下
1.可以自定义生成联系人选项<br />
2.可以自定义每个选项的个数或者最大个数和最小个数<br />
3.可以自定生成联系人个数<br />
4.可以生成完全重复联系人<br />
5.显示联系人列表和查看、搜索联系人

#如何使用
用android studio导入即可，注意gradle的版本，可以自己修改成支持的版本

#使用中会遇到的问题
如果无法生成联系人，该问题主要是由权限导致，可以查看是否有读写联系人权限
